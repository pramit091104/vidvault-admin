CREDENTIALS TO ROTATE (if already exposed):
===========================================
If any of these were committed to Git, you MUST rotate them:

1. GCS Service Account Key:

   - IAM & Admin → Service Accounts
   - Delete old key, create new one
   - Update in Render/Railway environment variables

2. Razorpay Keys:
   - Go to Razorpay Dashboard
   - Settings → API Keys
   - Regenerate keys
   - Update in environment variables

3. Gmail App Password:
   - Go to Google Account
   - Security → App Passwords
   - Revoke old password
   - Generate new one
   - Update in environment variables

4. Razorpay Webhook Secret:
   - Go to Razorpay Dashboard
   - Settings → Webhooks
   - Regenerate secret
   - Update in environment variables
